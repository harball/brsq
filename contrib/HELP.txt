                     FLIGHTGEAR "BOURRASQUE" AIRCRAFT - HELP


memo
====

keyboard shortcuts
------------------

- c : open/close canopy, you can click on the canopy handles or on the canopy switch to half-open it
- k : Extend Speedbrakes (5 positions)
- j : Retract Speedbrakes (5 positions)
- s : Toggle Autotrim (pitch)
- ctrl-a : enable autopilot on Altitude
- ctrl-s : enable autopilot on Speed
- ctrl-h : enable autopilot on Heading (bug)
- ctrl-n : enable autopilot on Heading (nav 1 radial)
- ctrl-p : enable autopilot to keep current heading, altitude and speed
- ctrl-P : disable autopilot
- m : select previous mod
- M : select next mod
- } : autostart
- v : next view
- V : previous view
- ctrl-v : default view (pilot)
- tab : mouse mode : pointer, view, stick
- < or \ : change phase for record loop
- > or | : reinit record loop
- [ : Catapult launch (carrier)
- ] : Toggle Launch-bar (carrier)
- ; : Toggle Hook (carrier)
- NUMPAD : see below


mouse functionnalities
----------------------

when on stick mode (cross cursor - hit TAB to change)
- pitch : vertical axis (no MMB pressed)
- roll : horizontal axis (no buttons pressed)
- yaw : horizontal axis (LMB or MMB pressed)
- throttle : vertical axis (MMB pressed)
- throttle : scroll UP/DOWN
- view : vertical/horizontal axis (RMB pressed)

instruments knobs behaviours
----------------------------

using knobs with mouse :
- LMB clic   = +10 or +30 or +100
- MMB clic   = -10 or -30 or -100
- scroll UP  = +1
- scroll DWN = -1

mods
----

4 buttons on the top of the center screen (Nav Display) change fly-by-wire, hud and ND settings
- TAXI : for taxiing
- APP : for landing, take-off and approach
- NAV : for navigation
- VFR : for free fly


how-to
======

in air start at 5000ft and 250kt
--------------------------------

launch flightgear adding 2 parameters : --altitude=5000 --vc=250
aircraft will automatically fast start and enable autopilot


fast start
----------

use the bourrasque menu


autostart
---------

use the bourrasque menu or } key


start
-----

FUEL TRUCK : click to adjust fuel quantity [ON DEMAND]
gui/equipment : fuel truck [REMOVE]
gui/equipment : covers [REMOVE]
gui/equipment : scale [REMOVE]
command panel : canopy [HALF-CLOSE]
electrical instrument : switch [BAT or EPU]
electrical instrument : avionics [ON]
lights panel : instrument light [ADJUST]
radio : NAV1 [ON]
radio : COM1 [ON]
radio : NAV2 [ON]
radio : COM2 [ON]
radio : ADF  [ON]
radio : TRANSPONDER [ON]
nav/com : frequency [ADJUST]

contact tower, request autorization to start engines

transponder adjust squawk [ON DEMAND]
electrical instrument : engines [ON]
engine 0 instrument : cut [ON]
engine 0 instrument : pump [ON]
engine 0 instrument : start [PUSH]
engine 0 started [WAIT]
electrical instrument : switch [ALT]
engine 1 instrument : cut [ON]
engine 1 instrument : pump [ON]
engine 1 instrument : start [PUSH]
lights panel : anti-collision [ON]
lights panel : strobe [ON]
lights panel : navigation [ON]
lights panel : position [ON]
lights panel : formation [ON DEMAND]
lights panel : landing and taxi [ON DEMAND]
engine 1 started [WAIT]
electrical instrument : cmd [ON]
gui/equipment : epu [REMOVE]

taxi
----

contact tower, request autorization to taxi

ND : mod [TAXI]
gui/equipment : shocks [REMOVE]
command panel : parkbrake [RELEASE]
brake [TEST]


take off
--------

contact tower, request autorization to enter on runway

command panel : canopy [CLOSE]
ND : mod [APP]
PFD : autopilot speed [ADJUST 200kt]
PFD : autopilot alt [ADJUST]
PFD : autopilot heading [ADJUST]
HSI : NAV1 radial [ADJUST]

full thrust
rotation: 150kts


navigation
----------

ND : mod [NAV]
ALT : hdg [1013 HPa]
choose VOR frequency, choose radial
ND : VOR1 [ENABLE]
PFD : adjust autopilot [ON DEMAND]


landing
-------

ALT : hdg [ADJUST]
nd : mod [APP]
approach at ~140kts use speedbrakes (j, k keys) (speedbrake 40% - aoa 5)
hud flight director on runway threshold
and hud flight director between -2deg and -4deg
touchdown at 120kts
nd : mod [TAXI]

carrier take off
----------------

nd : mod [APP]
engage launch bar (])
unlock parkbrakes (B)
throttle 80%, catapult ([)


refuelling as a tanker
----------------------

an automatic hippodrome driven by autopilot is available

gui/equipment : center pylon [refuel pod]
the copilot proGo view will be disable and the view refuelling pod will be enabled

PFD : autopilot speed [ADJUST to 200kt AND ENABLE]
PFD : autopilot alt [ADJUST to 10000ft AND ENABLE]
PFD : autopilot heading [ADJUST AND ENABLE]
wait aircraft is stabilized
PFD : autopilot hippodrome [ENABLE]
SFD : aar [ENABLE] (extend the refuelling pod pipe)
